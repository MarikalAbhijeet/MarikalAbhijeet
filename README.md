<h2>🚀 Featured AI Security Project</h2>

<h3>TraceAI Engine — Local-first AI SOC Investigation Platform</h3>

<p>
<b>AI security investigation, triage, evidence correlation, and human-reviewed reporting</b>
</p>

<p>
TraceAI Engine is a local-first AI security investigation workspace that combines deterministic SOC logic, Ollama-powered local LLM workflows, RAG-style case context retrieval, AI-assisted triage, evidence correlation, interactive relationship graphs, and Markdown/PDF incident reporting.
</p>

<ul>
  <li>Local LLM workflows with Ollama</li>
  <li>RAG-style SecOps Copilot over local case context and playbooks</li>
  <li>AI-assisted phishing and endpoint evidence triage</li>
  <li>Interactive evidence relationship graph using PyVis / NetworkX</li>
  <li>Deterministic + local AI routing for faster analyst workflows</li>
  <li>Case persistence using summarized evidence only</li>
  <li>Human-reviewed incident reports with MITRE ATT&CK and KQL pivots</li>
</ul>
